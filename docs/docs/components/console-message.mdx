# ConsoleMessage

Console messages allows you to display data like the Chrome console.

## Limitations

Currently only the following types are supported:
* `string`
* `number`
* `boolean`
* `object`
* `array`
* `undefined`
* `null`
* `function`

## Examples

### Simple message

<AsideLive
  code={
`<ConsoleMessage value={{count: 0}} />`
  }
/>

### Nested message

<AsideLive
  code={
`function Devtools() {
  const value = {
    nested: {
      deeplyNested: {
        count: 0,
      },
    },
  };

  return <ConsoleMessage value={value} />;
}`
}
/>

### Message with arrays

<AsideLive
  code={
`function App() {
  const value = {
    todos: [
      {id: 1, text: 'Learn React'},
      {id: 2, text: 'Learn Signals'},
      {id: 3, text: 'Learn Next.js'},
    ]
  };

  return <ConsoleMessage value={value} />;
}`
    }
/>
